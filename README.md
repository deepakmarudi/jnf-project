# JNF Portal

A campus recruitment system where recruiters submit job details and the placement team manages approvals.

## Tech Stack

### Frontend
- Next.js
- MUI
- NextAuth.js
- Node.js and npm

### Backend
- Laravel
- MySQL or MariaDB
- PHP 8.2 via XAMPP
- Composer

### Other Tools
- Git
- Postman
- HeidiSQL

## Start Here
