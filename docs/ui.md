# UI

## Objective

Create a recruiter/admin portal UI that feels institutional, modern, and easy to use for long form workflows.

## Design direction

- Formal but modern
- Calm and structured
- Enterprise SaaS with academic trust
- Avoid generic noisy dashboards

## Visual system

### Colors

- Primary blue: `#0b4f8a`
- Secondary copper: `#8b5e34`
- Background: `#f4f7fb`
- Text primary: `#0f172a`
- Text secondary: `#52607a`

### Typography

- Display: `DM Serif Display`
- Body/UI: `Manrope`

### Components

- Rounded cards
- Pill buttons
- Section chips
- Large form sections
- Status chips

### Public

- Admin dashboard

- Admin JNF detail
- Admin recruiter management

## UX priorities

- Make long forms feel manageable
- Reduce confusion with clear grouping
- Keep actions obvious
- Support mobile layouts cleanly
- Maintain strong readability
